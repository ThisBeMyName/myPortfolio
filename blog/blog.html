React

TLDR - What is React? How do I use it? What the blast am I even doing??

They say 'If You’re Feeling Uncomfortable, That’s when growth happens'. I think that's a feeling I've been accepting. I don't think you really get used to it, but it maybe more something you forget is there, or it ebbs the more you grow, then you forget how strong it felt in the beginning.

It always feels like that when I dive into a new technology. That feeling of uneasiness. Maybe I'm painting this in too much of a harsh light. It's not that it's bad, it's more of a question 'Where do I begin?'. There's always my trusty tools I go to.. Youtube, Google, code forums. Thus was how my journey into React started.

On a basic level, I knew what it was. A Javascript framework, right? It's one of those keywords you hear at some point in tech. As some point though, you just have to dive right in. You can watch all you want how to ride a bike, but until you get one one and bomb down a hill (as in my case), you'll never learn how to do it.

Random Quote Generator

TLDR - Generates random bodybuilding quotes using an API. Ran into issues incorporating the API data, but was able to work throug them pretty easy

This one was a pretty fun project to work on. Most of it was pretty strait forward. I ran into most of my issues when trying to implement the API portion. My first issue was I needed a key, but that was pretty easy to work around. Next, after running several tests, I kept getting a "Cors" error. I found a few workarounds for this one, but eventually shifted to another API (originally it was an API with motivational quotes, but I found another that had bodybuilding quotes I liked better). I didn't run into the error with this new API.

The hard part was pulling the data I needed from the response. After digging through the json a bit, I found the correct endpoints. At first, I felt a bit lost.. but in the sense that I was blind and knew braille (I knew how to use an API) and was given a map in braille. I had to comb over it several times until I got a bit more familure with it and could find my way around the data.

Once I had this, plugging in the peaces was pretty easy. I did run into a few other minor issues, but most of it was due to typos in the code.
